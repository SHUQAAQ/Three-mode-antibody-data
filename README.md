# Three-mode-antibody-data
We have compiled a three mode antibody dataset

We have sorted out a three mode dataset of antibodies from five aspects: antibody specificity, chemical properties, functional description, protein family and structural biology, respectively, including structural information (in pdb format), sequence and text description
